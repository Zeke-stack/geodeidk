## Installation

1. Install [Geode Mod Loader](https://geode-sdk.org/)
2. Download the latest release of FakeGDMod
3. Place the .geode file in your mods folder
4. Launch Geometry Dash and configure your role in mod settings

## Configuration

Access mod settings in-game to customize:

- **Role Type** - Choose between Moderator, Elder Moderator, Leaderboard Moderator, or Developer
- **Enable Level Featuring** - Toggle level featuring capability
- **Enable Epic Rating** - Toggle epic level marking
- **Enable Creator Points** - Toggle creator points assignment
- **Show Level Passwords** - View passwords for copy-protected levels  
- **Enable Comment Moderation** - Delete any comments
- **Glowing Username** - Enable username glow effect (Developer only)
- **Unlock All Icons** - Access all icons and colors
- **Bypass Rate Limits** - Skip cooldown timers

## Usage

1. Go to the Support menu in Geometry Dash
2. Request moderator/developer access
3. Your selected role will be activated
4. Use your new privileges throughout the game!
